You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the results of two numbers multiplied
  • /temperature (ex. /temperature?value=32&unit=C) -- This request converts temperature from one unit to another.
  • /wordcount (ex. /wordcount?text=Hello%20world%20and%20welcome) -- This request counts the number of words in a given text
